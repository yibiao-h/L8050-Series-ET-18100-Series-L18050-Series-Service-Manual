# Disassembly Flowchart

This section describes procedures for disassembling the parts/units in a flowchart format.

Understand how to read the flowchart, which is described below, and follow the flowchart to disassemble and reassemble the product. 

When removing major components, also see the "Disassembly & Reassembly" section.

When routing cables, see the Routing FFCs/cables".

#How to Read the Disassembly Flowchart

The following explains how to read the disassembly flowchart using an example one.

No.Description
1Parts/units with black background are provided as ASP. Click on an underlined one to jump to the disassembly procedure for that part/unit. 
2Indicates the number of screws securing the parts/units.
3Indicates how many hooks, ribs, dowels, etc. are used to secure the part/unit other than screws.
4

Indicates the work required to proceed to the next step.

If only a part name is shown, remove that part; if an FFC or cable name is shown, disconnect them from the connector.

5

Parts/units with gray background are those that need to be removed in order to remove the target part/unit.

6

Indicates screw types that are defined in "List of Screw Types".

7Links to the other flowchart.
8Parts/units with white background are not provided as ASP. Click on an underlined one to jump to the disassembly procedure for that part/unit. 
9Indicates that lubrication is required when replacing the part/unit. Click on the icon to jump to the lubrication page.
10Indicates that adjustment is required when replacing the part/unit. Click on the icon to jump to the adjustment page.
11Links to the "List of Screw Types" page. If necessary, right-click it and select "Open link in new tab".
